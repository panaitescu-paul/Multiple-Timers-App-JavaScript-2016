# Multiple Timers App - JavaScript - 2016
### Operate Multiple Timers at once.

## Features:
1. **Set Timer**
    - Title
    - Description
    - Hours, minutes, seconds
2. **Start/Stop Timer**
3. **Operate multiple Timers**

## Technologies:
1. Angular JS
2. JavaScript
3. Google Fonts
4. HTML
5. CSS

## Tools:
- Sublime Text
